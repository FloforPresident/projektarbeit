USER:

- DELETE USER (name, password)
- LOGIN USER (name, password)
- LOGOUT USER (name)

ROBO:
- ADD ROBO (name, ?ip?, ?username?)
- DELETE ROBO (name, ?username?)

ROOM:
- ADD ROOM (name, roboname, ?username?) -> weiterverarbeitung im socket für room scanning
- DELETE ROOM (name, ?username?)
- ?EDIT ROOM? (name, ?username?)

LOCATION:
- ADD LOCATION (name, x, y, username, ?room o. robo?) -> weiterverarbeitung im socket wegpunkt auf map markieren
- DELETE LOCATION (name, ?username?)
- CHANGE ACTIVE LOCATION (locationname, username)

MESSAGE:
- SEND MESSAGE (username - sender, username - receiver, text) -> weiterverarbeitung im socket (Robo x fahr zu Person y mit location z)
